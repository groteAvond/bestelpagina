De bestanden bestellingGA.css, bestellingGA.js en bestellingGA.php zijn alle code voor de plugin bestellingGA in Wordpress.
Het bestand index.ts is alle code voor de Edge Function van de Supabase database bestellingenGA.
Ook is er een handleiding toegevoegd om de belangrijkste dingen aan te passen op de website en in de database. Deze handleiding is voor zowel de website als het programma Zaalindeler.
